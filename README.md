# Frame. A Home for Every Lens.

Frame is a sophisticated, photography-focused web platform designed to prioritize minimalism and high-fidelity visual storytelling. It provides a clean, distraction-free environment for photographers to showcase their portfolios and exhibitions.

## Features

- **Modern Editorial Design**: Implements the "Luminous Editorial" design system with elegant typography (Playfair Display & Inter) and a sleek light-themed color palette.
- **Responsive Layout**: Fully responsive experience across desktop, tablet, and mobile devices.
- **Gallery Carousel**: A dedicated gallery page (`/gallery`) for browsing high-quality photography.
- **Client-Side Routing**: Fast, seamless navigation using React Router.
- **Premium Aesthetics**: Utilizes glassmorphism (backdrop blurs), smooth transitions, and subtle hover effects for a highly polished feel.

## Tech Stack

- **Framework**: [React 19](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling & UI**: [Material UI (MUI)](https://mui.com/) & [Emotion](https://emotion.sh/)
- **Routing**: [React Router](https://reactrouter.com/)

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd project-photo
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and visit `http://localhost:5173` (or the port specified in your terminal).

## Project Structure

- `src/components/`: Modular UI components (NavBar, Hero, FeatureCards, GalleryCarousel, Footer).
- `src/App.tsx`: Main application file containing routing logic.
- `src/index.css`: Global CSS and CSS variables for the design system.
- `DESIGN.md` / `AltDesign.md`: Documentation for the core design system tokens and guidelines.

## Design System

`!!! These design are generated by Google Stitch AI.`

Frame utilizes a curated set of foundational design tokens:
- **Typography**: Playfair Display (Headlines), Inter (UI elements)
- **Colors**: 
  - Background: Minimalist light gray (`#F9F9F9`)
  - Primary Text/Accents: Stark Black (`#000000`) & Deep Gray (`#1A1C1E`)
  - Borders: Subtle translucent black (`rgba(0, 0, 0, 0.1)`)
- **Components**: UI elements feature sharp corners (`0px` border radius) and clean outlines to match the sophisticated editorial vibe.
